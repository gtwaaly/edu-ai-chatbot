# 🎓 教育培训 AI 智能客服系统

> 基于 Xiaomi MiMo API 构建的教育培训场景 AI 问答机器人，支持私有知识库上传与多轮对话。

---

## 📌 项目背景

在教育培训行业，学员咨询量大、问题重复率高，人工客服每天需花费大量时间回答"课程有哪些"、"怎么报名"、"多少钱"等基础问题。

本项目旨在解决这一痛点：**让机构上传自己的课程资料，AI 自动回答学员咨询，解放人工客服时间。**

---

## ✨ 功能特性

- 📎 **私有知识库** — 上传机构课程文档（txt/md），AI 基于文档内容回答问题
- 💬 **多轮对话** — 保留完整对话历史，回答上下文连贯
- ⚡ **快捷提问** — 内置常见问题快捷按钮，学员一键提问
- 🔑 **安全存储** — API Key 仅存储在本地浏览器，不上传任何服务器
- 📱 **零部署** — 纯前端实现，无需后端，打开 HTML 文件即可使用

---

## 🛠 技术架构

```
用户输入问题
    ↓
读取本地知识库文档内容
    ↓
构建 System Prompt（文档内容 + 角色设定）
    ↓
调用 Xiaomi MiMo API（mimo-v2-pro）
    ↓
返回回答，更新对话历史
    ↓
展示给用户
```

**技术栈：**
- 前端：HTML5 + CSS3 + Vanilla JavaScript
- AI 模型：Xiaomi MiMo API（mimo-v2-pro）
- 知识库方案：RAG（检索增强生成）思路，文档注入上下文

---

## 🚀 快速使用

**1. 克隆或下载项目**
```bash
git clone https://github.com/gtwaaly/edu-ai-chatbot.git
```

**2. 直接用浏览器打开 `index.html`**

**3. 输入你的 MiMo API Key**
- 首次打开会弹出输入框
- 前往 [platform.xiaomimimo.com](https://platform.xiaomimimo.com) 获取 API Key

**4. 上传你的课程文档**
- 点击"上传文档"按钮
- 支持 txt、md 格式
- 可上传多个文档

**5. 开始对话！**
